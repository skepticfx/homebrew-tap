# skepticfx/homebrew-tap

Homebrew tap for `sole`.

## Install

```bash
brew install skepticfx/tap/sole
```

This also installs `Sole.app` automatically so you can launch it like a normal macOS app.
The installer prefers `/Applications/Sole.app` and falls back to `~/Applications/Sole.app`.

Or tap first:

```bash
brew tap skepticfx/tap
brew install sole
```
